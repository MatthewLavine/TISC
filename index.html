💡 The Big Idea

A CPU does one thing over and over, billions of times per second:

📥
Fetch
Read the next instruction from memory
→
🔍
Decode
Figure out what the instruction means
→
⚙️
Execute
Perform the operation
↩

This is the fetch–decode–execute cycle. Every program you've ever run — from video games to web browsers — is just this loop running on different sequences of instructions.
